# Shlinkly

Native iOS client for self-hosted [Shlink](https://shlink.io/) URL shortener.

**Status:** 🚧 Early development (Phase 0)

## What

Open-source SwiftUI app that connects to your own Shlink instance. Manage short URLs, view analytics, integrate with iOS Share Sheet and Shortcuts — all from your iPhone.

## Why

Self-hosters running Shlink need a clean native iOS client. Shlinkly aims to be one option alongside [Shlink Manager](https://apps.apple.com/us/app/shlink-manager/id6758879156) with a focus on open source + deeper Apple ecosystem integration (App Intents, Spotlight, Lockscreen Widget).

## Status

In active development. Roadmap and decisions tracked privately for now; public roadmap will appear here closer to TestFlight beta.
